Fast paced beacon visualization

Visualization of fast paced beacon, more info on specialized beaconing setup from RIPE NCC.

  • Currently using the ipv4 beacon 84.205.66.0/24. Limited to maximum path length = 3 by default (or query with ?maxPathLen=N)
  • Announcements times at : XX:10, XX:30, XX:50, withdrawals times at : XX:00, XX:20, XX:40.
  • You'd likely need to wait until announcement times to see any activities (and withdrawals 10 mins later).
